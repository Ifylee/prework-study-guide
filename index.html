Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • We use git add -A to stage a change.
  • git commit -m "description" is used to commit the changes to our repo
  • Then git pull origin main to make sure our current branch is up to date before opening a pull request
  • git push origin feature/add-css to push up all our changes
  • Then we go to Github repo and pull new request.
  • Then we merge pull request and close the issue for the topic
  • We use git checkout main to update the local main branch
  • Lastly, we use the git pull origin main to pull the remote [main] branch into the local[main] branch.
