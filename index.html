HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is the first script to be written in any web development
  • It is the main structure of the webpage just like in building a house. HTML is the frame work.
  • The header serves like the foundation of the house. It is not visible to the web users.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Css is the ecstatic part of the web page. It is where you beautify your web with all the colors and attributes you want users to see.
  • the Css page is added after the last meta at the head before the title .
  • Borders,lines,font types and colors etc are all set on the css script.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • We use git add -A to stage a change.
  • git commit -m "description" is used to commit the changes to our repo
  • Then git pull origin main to make sure our current branch is up to date before opening a pull request
  • git push origin feature/add-css to push up all our changes
  • Then we go to Github repo and pull new request.
  • Then we merge pull request and close the issue for the topic
  • We use git checkout main to update the local main branch
  • Lastly, we use the git pull origin main to pull the remote [main] branch into the local[main] branch.
